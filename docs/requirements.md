# Requirements

## Supported Platform
- macOS host (Darwin)

## Host Requirements
- Built-in macOS tools:
  - `diskutil`
  - `plutil`

## Build Requirements
- Go toolchain in `PATH`

## Container Requirements
- Docker Desktop
- `docker compose`
- Ventoy version is pinned in `docker/Dockerfile`
- Ventoy tooling container runs as `linux/amd64`
